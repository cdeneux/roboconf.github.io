Outils

Outillage

Au-delà de la façon dont Roboconf est implémenté, Roboconf a été pensé pour être aussi simple que possible. Cette question de la simplicité se retrouve dans la syntaxe des fichiers de configuration.

Même si certains de ces choix ont été faits de sorte à ne pas dépendre d'un outillage, il s'avère que des outils sont toujours utiles.
Ils aident notamment de nouveaux utilisateurs, et plus généralement, ils permettent de gagner du temps.

L'Administration Web - 1/2

L'administration web est, comme son nom l'indique, une application web (AngularJS) qui fournit une interface graphique pour l'API REST du DM.

L'Administration Web - 2/2

Cette console permet de donner des ordres aux agents Roboconf et d'avoir une vue globale des différentes parties applicatives.

Plug-in Maven

Roboconf possède son propre plug-in Maven.
Il permet...

  • ... de valider ses fichiers de configuration (graph, instances)...
  • ... de conditionner son projet sous la forme d'une archive déployable...
  • ... d'insérer des propriétés Maven dans ses fichiers de configuration.


Des travaux sont en cours pour faciliter les tests et la réutilisation de recettes.
